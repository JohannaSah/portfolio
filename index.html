Chat App

React Native, Gifted Chat, Expo

Chat App is an application for both Android /iOS created with Reactive Native and Expo. The app provides users with a customizable chat interface and options to take pictures, share images and their location. This app has Google Cloud Firestore as a non-relational database where images and messages are being stored. This last feature allows users to read chat messages even when the app is offline.

Check out the apps functionality in this video
and feel free to check out the code on github .

Meet App - PWA

React, Google Calendar API, Google Auth

This is a React application that allows users to view and schedule events from a Google Calendar. It utilizes the Google Calendar API to retrieve data and presents it in a card format. This API is protected and requires authorization access. Users can click on a card to access more details about an event and can also search for a specific event by name or number. MeetApp is built using a test-driven development technique (TDD) It's designed as a serverless, progressive web app (PWA).

Check out the apps functionality in this video
and feel free to check out the code on github .

MyFlix - Api

JavaScript, MongoDB, Express, Node.Js, Render

This is a REST API that allows users to access information about movies, directors, and genres. Users can also create an account, update their information, and add movies to their list of favorites. It utilizes the MERN stack (MongoDB, Express, React, Node.js) and is deployed to Render.

Feel free to check out the code on github .

MyFlix - React

HTML, CSS, JavaScript, React, Bootstrap, Parcel

Myflix is a single page application built with React that allows users to register an account, log in, and browse a collection of movies. Users can also view information about a movie, including its director and genre, and add movies to their list of favorites. This project was built using React, React Redux, and React Bootstrap. It uses Parcel as its built tool and it is hosted on Netflify.

Feel free to check out live site and the code on github .

MyFlix - Angular

HTML, CSS, JavaScript, Angluar, Node.js

The client side of a movie browsing app that allows users to register, log in and browse through movies. The users can also add movies to their list of favourites and view or update their profile details. Built with Angular and designed with Angular Material.

Feel free to check out live site and the code on github .

To-do List

HTML, CSS, jQuery

This app allows the user to create a to-do list. Items can be added, crossed out, rearranged and deleted. Built using HTML, CSS and jQuery.

Feel free to check out live site and the code on github .

Meet App - PWA

React, Google Calendar API, Google Auth

Dies ist eine React-Anwendung, mit der Benutzer Ereignisse aus einem Google Kalender anzeigen und planen können. Sie verwendet die Google Calendar API, um Daten abzurufen und präsentiert sie in Kartenformat. Diese API ist geschützt und erfordert autorisierten Zugriff. Benutzer können auf eine Karte klicken, um weitere Details zu einem Ereignis anzuzeigen, und können auch nach einem bestimmten Ereignis nach Namen oder Nummer suchen. MeetApp wurde unter Verwendung der testgesteuerten Entwicklungstechnik (TDD) entwickelt. Es ist als serverlose, progressive Webanwendung (PWA) konzipiert.

und werft einen Blick auf den Code auf Github .
